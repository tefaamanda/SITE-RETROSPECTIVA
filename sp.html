Sites integrados com Python

Recentemente, começamos a trabalhar sites com HTML e CSS integrados com a parte do back-end programada com uso do PyCharm. Ainda sinto bastante dificuldades ao programar pelo PyCharm mas é também uma experiência muito bacana.



Conhecimentos Adquiridos

  • Programação com linguagem em Python no PyCharm.

  • Cabeçalho com links declarados no "main.py".

  • Instalação do Flask.

  • Integração de Front-End e Back-End.

  • Armazenar dados de formulários em listas.



Dificuldades

  • Identificar o erro encontrado na programação.

  • Conectar as rotas em uma única página.
